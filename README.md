# ATM CLI

A simple command-line ATM application.

## Getting Started

1. **Install dependencies:**
   ```bash
   npm install
   ```

2. **Start the application:**
   ```bash
   npm run start
   ```

3. **Run tests:**
   ```bash
   npm test
   ```

Or use the provided scripts:

- To start: `./start.sh`
- To test: `./test.sh`
